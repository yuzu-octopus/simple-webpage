About

simple-webpage is a vanilla HTML/CSS/JS reference repository and reusable OpenCode agent skill. No frameworks, no build steps, no bloat.

Inspired by 37signals (Campfire/Writebook), Coding2GO, and the ponytail YAGNI philosophy.

Getting Started

  • 1. Browse the cookbook — Open any file in skills/simple-webpage/cookbook/ in your browser.
  • 2. Copy patterns — Grab the HTML/CSS you need and paste into your project.
  • 3. Customize tokens — Override CSS custom properties from tokens/ to match your brand.
  • 4. Load the skill — Add SKILL.md to your OpenCode config for AI-assisted development.
