Fluid Typography with clamp()

Resize the browser window. The heading scales smoothly between 30px and 48px without any media queries.

How it works

clamp(1.875rem, 4vw + 1rem, 3rem) sets a minimum, preferred, and maximum font size. The browser interpolates between them based on viewport width.

When to use

For any text that should scale with the viewport — headings, hero text, body copy. Replace multiple media query breakpoints with one line.
